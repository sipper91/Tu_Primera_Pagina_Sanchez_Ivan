#🛠️ Ferretería y Barraca La Fe - Proyecto Final

Este es el proyecto final para el curso de Python de la mano de CODERHOUSE. Consiste en una aplicación web desarrollada con **Django** que integra la gestión de una ferretería con un sistema de blog para noticias.


##  Alumno 
- **Ivan Sanchez** - Curso Python Flex (Coderhouse)

##  Funciones Principales
- **Blog (Pages):** Despliegue de noticias con imágenes y texto (CKEditor).
- **Inventario:** Sistema unificado de listado, búsqueda por nombre y eliminación de productos para usuarios logueados.
- **Proveedores:** Directorio con gestión de contactos y rubros.
- **Usuarios (Accounts):** Registro, Login, Perfil personalizado y sistema de Avatares.

##  Herramientas
- Python 3.13 / Django 6.0.2
- SQLite (Base de Datos)
- CKEditor (Texto Enriquecido)
- Pillow (Manejo de Imágenes)

## Instalación
1. Clonar el repositorio.
2. Instalar dependencias: `pip install -r requirements.txt`dd
3. Ejecutar migraciones: `python manage.py migrate`
4. Iniciar servidor: `python manage.py runserver`
